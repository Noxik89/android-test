# Network Frequency Controller

Android приложение для определения и управления частотами мобильной сети (3G, 4G LTE).

## Описание

Приложение предназначено для тестирования мобильных сетей. Позволяет:
- Определять текущие частоты мобильного оператора
- Просматривать информацию о подключённой сети
- Отображать доступные частоты в реальном времени
- Логировать все изменения частот с историей

## Технические характеристики

- **Минимальная версия Android:** 14+ (API level 34+)
- **Поддерживаемые сети:** 3G (WCDMA), 4G (LTE), 5G (NR)
- **Язык программирования:** Kotlin
- **Архитектура:** MVVM с Android Architecture Components
- **База данных:** Room для хранения истории

## Требуемые разрешения

- `READ_PHONE_STATE` - чтение состояния телефона
- `ACCESS_NETWORK_STATE` - доступ к информации о сети
- `READ_PRECISE_PHONE_STATE` - точная информация о состоянии телефона (Android 10+)

## Функциональность

### Основной экран
- Отображение информации об операторе
- Тип сети (3G/4G/5G)
- Текущая частота и диапазон
- Список доступных частот с уровнем сигнала
- Кнопка обновления информации
- Кнопка просмотра истории

### История изменений
- Полная история всех попыток изменения частот
- Временные метки
- Старая и новая частота
- Статус операции
- Информация об операторе и типе сети

## Сборка проекта

### Требования
- Android Studio Hedgehog | 2023.1.1 или новее
- JDK 17
- Android SDK 34+

### Команды сборки

```bash
# Сборка debug APK
./gradlew assembleDebug

# Сборка release APK
./gradlew assembleRelease

# Установка на подключённое устройство
./gradlew installDebug
```

APK файлы будут находиться в:
- Debug: `app/build/outputs/apk/debug/app-debug.apk`
- Release: `app/build/outputs/apk/release/app-release-unsigned.apk`

## Ограничения

⚠️ **Важно:** Изменение частот программно ограничено в Android без root-доступа. 

Приложение может:
- ✅ Определять текущие частоты и параметры сети
- ✅ Отображать доступные диапазоны и частоты
- ✅ Логировать попытки изменения частот
- ❌ Напрямую изменять частоты без root-прав (ограничение платформы Android)

Для реального переключения частот пользователю необходимо:
1. Использовать инженерное меню устройства (зависит от производителя)
2. Получить root-доступ
3. Использовать специальные команды оператора

## Использование

1. Установите APK на устройство с Android 14+
2. При первом запуске предоставьте необходимые разрешения
3. Приложение автоматически определит текущие параметры сети
4. Нажмите на частоту в списке для просмотра деталей
5. Используйте кнопку обновления для получения актуальных данных
6. Откройте историю для просмотра всех логов
